# Hanyang_ERICA_Drinkspot
신입생을 위한 한양 에리카 근처(도보 20분 이내) 술집 조사 및 분석 프로젝트
 1) 조원역할
  1-1) 김은호 - 조장, 써클, 투다리, 락포차, 한양사잇길, 역전할머니, 에이플러스 index 수정 및 작성, READ ME 업로드
  1-2) 김누리 - 페이지 전체 디자인 생성, 펀비어킹, 교반, 더술 자료조사 및 index 수정, 작성
  1-3) 마효빈 - 회의록 작성, 페이지 전체 디자인 생성 보조, 만취, 로드락비어, 달달포차 자료조사 및 index 수정, 작성
  1-4) 이정우 - READ ME 내용 작성, 써클, 투다리, 락포차, 한양사잇길, 역전할머니, 에이플러스 자료조사
 2) 자료 조사 방법
  : 네이버 지도 후기 및 직접 방문
 3) AI 도움 받은 요소
  : 전체 레이아웃 제작에서 도움을 받았으며 이후 색상과 폰트 수정과정을 직접 진행함
